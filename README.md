# React Web Apps

Welcome to the **React Web Apps** repository! This project hosts a collection of practical, modular web applications built using **React.js**. It is designed to demonstrate fundamental to intermediate React concepts, including component architecture, state management, hooks, and third-party API integration.

---

## 📂 Repository Structure

This repository contains the following mini-applications:

| Folder Name | Description | Key Technical Features |
| :--- | :--- | :--- |
| 🔢 **`Counter`** | A classic state-controlled counter application. | React `useState` hook, event handling, and conditional styling. |
| 🎬 **`Movie Search App`** | An application to browse and search for movies. | Fetching data from external movie APIs (e.g., OMDB/TMDB), search debouncing, and rendering dynamic lists. |
| 🍳 **`Recipe App`** | A culinary search tool for discovering recipes. | Async API requests, parsing complex JSON objects, and layout formatting for ingredient lists. |
| ☀️ **`Weather App`** | A real-time weather tracking application. | Location-based API fetching (e.g., OpenWeatherMap), parsing weather metrics, and dynamic background updates based on current weather conditions. |

---

## 🛠️ Built With

* **React.js** (Functional Components & Hooks)
* **JavaScript (ES6+)**
* **CSS3 / HTML5**
* **REST APIs** (for dynamic data fetching)

---

## 🚀 Getting Started

To run any of these applications locally, ensure you have [Node.js](https://nodejs.org/) installed on your machine.
